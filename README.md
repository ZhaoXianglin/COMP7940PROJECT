## Automatic Deployment
download the docker folder
>cd comp7940_mysql

create image comp7940_mysql

> docker build . -t comp7940_mysql

open folder comp7940_chatbot
>cd comp7940_chatbot

create image comp7940_chatbot

> docker build . -t comp7940_chatbot

run the servers

>  docker-compose up -d

enjoy it
